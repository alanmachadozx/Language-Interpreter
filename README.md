#    Language-Interpreter

This is a C++ project for a language interpreter, intended for study purposes. It is still under development, so you may find unimplemented features.

#  Struct

- Lexer (Scanner): Converts the source code into a sequence of tokens.
- Parser: Transforms tokens into an AST (Abstract Syntax Tree).

# How execute
```
# Clone repo 
git clone https://github.com/alanmachadozx/Language-Interpreter
cd Language-Interpreter

# Build
mkdir build && cd build
cmake ..
make

```
